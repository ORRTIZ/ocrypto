# ocrypto
3rd party FINTECH integration component for ORRTIZ™ BMS and Apache OFBiz for cryptocurrency; as a separate and optional hot-deploy component. 
This component enables you to configure your cryptocurrency integration and provides templates for inclusion in your OFBiz&trade; web store.
